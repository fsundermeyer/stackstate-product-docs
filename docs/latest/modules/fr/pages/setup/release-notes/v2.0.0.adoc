== Notes de version StackState version 7.0.0-snapshot.20240911112250-master-f9361e0 Helm Chart version 2.0.0

=== Nouvelles fonctionnalités et améliorations :

Il s'agit de la première version de SUSE Observability.
Il a les mêmes fonctionnalités que StackState helm chart version 1.11.4 mais avec une image de marque améliorée et une mise à jour de HBase 2.5.

Il n'est pas (encore) possible de mettre à jour StackState 1.11.4 vers SUSE Observability.

=== Changements :

* Le bouton "Show logs" sur la page de mise en évidence des pods est supprimé. Les journaux sont toujours accessibles via le lien "Voir tout" sur le widget des journaux qui se trouve sur la même page des faits saillants.
