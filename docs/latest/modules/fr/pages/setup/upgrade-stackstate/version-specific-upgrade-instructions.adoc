Cette page fournit des instructions spécifiques et des détails sur les étapes manuelles nécessaires à la mise à niveau vers chaque version prise en charge de SUSE Observability. *Lisez et appliquez toutes les instructions de la version que vous utilisez actuellement jusqu'à la version vers laquelle vous allez passer.*
