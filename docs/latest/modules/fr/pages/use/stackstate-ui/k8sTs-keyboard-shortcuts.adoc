= Raccourcis clavier
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Les raccourcis clavier ci-dessous peuvent être utilisés pour naviguer dans l'interface utilisateur de SUSE Observability.

|===
| Raccourci clavier | Action

| `CTRL` + `SHIFT` + `F`
| Rechercher un composant par son nom dans la xref:/use/views/k8s-topology-perspective.adoc[perspective topologique].

| `CTRL` + `S`
| Écraser les paramètres précédemment enregistrés pour une vue. Cette opération permet de mettre à jour la vue sauvegardée afin d'y inclure les modifications apportées aux xref:/use/views/k8s-filters.adoc[filtres de vue] ou aux xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation].

| `CTRL` + `SHIFT` + `S`
| Enregistrer les paramètres de la vue actuelle dans une nouvelle vue. La nouvelle vue comprendra tous les xref:/use/views/k8s-filters.adoc[filtres de vue] et les xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation] actuellement appliqués.

| `CTRL` + `SHIFT` + `R`
| Réinitialiser toutes les modifications non enregistrées et revenir à la vue, à la sous-vue ou au mode d'exploration d'origine. Cette opération réinitialise toutes les modifications non enregistrées apportées aux xref:/use/views/k8s-filters.adoc[filtres d'affichage] ou aux xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation].

| `SHIFT` + `1`
| Dans une vue, une sous-vue ou en mode exploration, ouvrez la xref:/use/views/k8s-topology-perspective.adoc[perspective topologique].

| `SHIFT` + `2`
| Dans une vue, une sous-vue ou en mode exploration, ouvrez la xref:/use/views/k8s-events-perspective.adoc[perspective Événements.]

| `SHIFT` + `3`
| Dans une vue, une sous-vue ou un mode d'exploration, ouvrez la xref:/use/views/k8s-metrics-perspective.adoc[perspective des métriques].

| `SHIFT` + `4`
| Dans une vue, une sous-vue ou en mode exploration, ouvrez la xref:/use/views/k8s-traces-perspective.adoc[perspective Traces].
|===
