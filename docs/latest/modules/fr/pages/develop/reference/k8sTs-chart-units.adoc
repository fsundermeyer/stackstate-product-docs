= Unités prises en charge pour les graphiques
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Cette page contient toutes les unités prises en charge par les cartes. Pour chaque unité, la description de l'unité est d'abord mentionnée, suivie du nom de l'unité qui devrait être utilisée dans les API de SUSE Observability. Par exemple, pour obtenir une valeur affichée en "notation scientifique", utilisez `sci` comme valeur de l'unité dans une liaison métrique.

Le rendu de l'unité s'adapte automatiquement à la taille de la valeur. Par exemple, si l'unité d'un graphique est `bytes`, la valeur est interprétée comme `bytes(IEC)`. Une valeur de `100` sera rendue par `100B`, mais une valeur de `5242880` (`5*1024*1024`) sera affichée par `5MiB`.
